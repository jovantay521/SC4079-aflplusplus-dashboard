Clone the repository
```
git clone https://github.com/jovantay521/SC4079-aflplusplus-dashboard
cd SC4079-aflplusplus-dashboard
```

Create python virtual environment
```
python -m venv .venv
```

Activate virtual environment
```
source .venv/bin/activate  # linux
.venv/scripts/activate  # windows
```

Run the app
```
streamlit run
```

For debug mode, export `DEBUG` variable:
```
export DEBUG=1
streamlit run
```

To deactivate virtual environment
```
deactivate
```
